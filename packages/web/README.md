## Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- [React](https://reactjs.org)

## Installation
Install the dependencies using:

```bash
yarn
```

## Usage
Run with developer mode using `dev`:

```bash
yarn dev
```

## Packaging
To generate a project package run `package`:

```bash
yarn package
```
